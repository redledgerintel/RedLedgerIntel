# Red Ledger Intelligence Engine

## Mission
Automated identification and verification of unlicensed gambling operators targeting regulated markets (UK, EU, LATAM).

## Core Capabilities
1. [cite_start]**The Dragnet:** Automated ingestion of "Black Market" search terms (e.g., "casinos not on gamstop")[cite: 4, 10].
2. **Verification Logic:** Cross-referencing identified URLs against:
   - [cite_start]UKGC Licensed Operator Register (Positive Match)[cite: 26].
   - [cite_start]Known Offshore Jurisdictions (Curacao, Anjouan) (Negative Match)[cite: 27].
3. [cite_start]**Asset Denial:** Outputting verified target lists for financial blocking and ISP enforcement[cite: 134, 140].

- `/targets_sample_v1.csv` - The Gold Standard data schema for target identification.
